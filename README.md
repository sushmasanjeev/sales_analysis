# 📈 Sales Data Analysis & Forecasting System

A complete sales analytics pipeline that analyzes past sales trends
and forecasts future revenue using machine learning models.

---

## 📌 Project Overview

This system helps businesses analyze sales performance and predict
future revenue using Linear Regression, Random Forest, and
Gradient Boosting models.

---

## 📂 Project Structure

sales_analysis/
├── data/                  # Dataset (retail_sales.csv)
├── notebooks/             # Jupyter notebook walkthrough
├── outputs/               # Charts and interactive dashboard
├── main.py                # Main pipeline script
└── requirements.txt       # Dependencies

---

## 🔧 Installation & Setup

# 1. Clone the repository
git clone https://github.com/yourusername/sales-analysis.git

# 2. Navigate to folder
cd sales-analysis

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the project
python main.py

- Exploratory Data Analysis (12+ charts)
